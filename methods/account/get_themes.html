GetThemesRequest

Both users and bots may be able to use this method. See code examples.

---functions---
account.getThemes#285946f8 format:string hash:int = account.Themes

Returns

account.Themes

This type can be an instance of either:

ThemesThemesNotModified

Parameters

formatstring
hashint

Known RPC errors

This request can't cause any RPC error as far as we know.

Example

const { TelegramClient } = require('gramjs');
const { functions, types } = require('gramjs/tl');

(async () => {
    const client = new TelegramClient(name, apiId, apiHash);
    await client.start();

    const result = await client.invoke(new functions.account.GetThemesRequest(
        format='some string here',
        hash=0
    ));
    console.log(result.stringify());
})();

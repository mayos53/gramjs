GetStringsRequest

Only users can use this method. See code examples.

---functions---
langpack.getStrings#efea3803 lang_pack:string lang_code:string keys:Vector<string> = Vector<LangPackString>

Returns

A list of the following type is returned.

LangPackString

This type can be an instance of either:

LangPackStringLangPackStringDeleted
LangPackStringPluralized

Parameters

lang_packstring
lang_codestring
keysstringA list must be supplied.

Known RPC errors

This request can cause 1 known error:

LangPackInvalidErrorThe provided language pack is invalid.

You can import these from gramjs/errors.

Example

const { TelegramClient } = require('gramjs');
const { functions, types } = require('gramjs/tl');

(async () => {
    const client = new TelegramClient(name, apiId, apiHash);
    await client.start();

    const result = await client.invoke(new functions.langpack.GetStringsRequest(
        lang_pack='',
        lang_code='en',
        keys=['some string here']
    ));
for x in result:
        print(x);
})();
